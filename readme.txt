How It Works:
Players take turns clicking on cells to place their marks (X or O) on the board.
The game checks for win conditions after each move to determine if there is a winner or if the game is a draw.
When the game ends, the status is updated to display the result.
Pros and Cons:
Pros:

Simple and intuitive user interface.
Provides basic Tic-Tac-Toe functionality.
Lightweight and fast.
Cons:

Limited features compared to more advanced Tic-Tac-Toe applications.
